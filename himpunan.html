01

Definisi Dasar Himpunan

Sebuah himpunan adalah kumpulan objek yang terdefinisi dengan jelas (well-defined), yang disebut elemen atau anggota. Teori himpunan menjadi fondasi dari seluruh matematika modern.

Definisi 1.1 — Himpunan
Himpunan \(A\) adalah koleksi benda-benda tertentu yang dianggap sebagai satu kesatuan. Jika \(x\) adalah elemen \(A\), maka ditulis \(x \in A\). Jika tidak, ditulis \(x \notin A\).

Notasi Himpunan

  • Notasi Daftar\(A = \{1, 2, 3, 4, 5\}\) — mendaftar semua anggota
  • Notasi Pembentuk\(A = \{x \mid P(x)\}\) — menggunakan predikat
  • Kardinalitas\(|A|\) atau \(n(A)\) — jumlah elemen dalam \(A\)

Himpunan-Himpunan Khusus

  • \(\emptyset\) atau \(\{\}\)Himpunan kosong — tidak memiliki anggota, \(|\emptyset| = 0\)
  • \(U\) atau \(S\)Himpunan semesta — memuat semua objek yang dibicarakan
  • \(\mathbb{N}\)Himpunan bilangan asli: \(\{0, 1, 2, 3, \ldots\}\)
  • \(\mathbb{Z}\)Himpunan bilangan bulat: \(\{\ldots, -2, -1, 0, 1, 2, \ldots\}\)
  • \(\mathbb{Q}\)Himpunan bilangan rasional
  • \(\mathbb{R}\)Himpunan bilangan real
Definisi 1.2 — Kesamaan Himpunan
Dua himpunan \(A\) dan \(B\) dikatakan sama, ditulis \(A = B\), jika dan hanya jika keduanya memiliki anggota yang persis sama. Secara formal: \(A = B \Leftrightarrow \forall x\, (x \in A \leftrightarrow x \in B)\).
Definisi 1.3 — Himpunan Bagian
\(A \subseteq B\) (A adalah himpunan bagian dari B) jika dan hanya jika setiap anggota \(A\) juga merupakan anggota \(B\). Secara formal: \(A \subseteq B \Leftrightarrow \forall x\, (x \in A \rightarrow x \in B)\).

02

Operasi Himpunan

Operasi pada himpunan menghasilkan himpunan baru dari himpunan yang ada. Ada empat operasi utama yang akan sering digunakan dalam teorema-teorema berikut.

Gabungan (Union)
\[A \cup B = \{x \mid x \in A \text{ atau } x \in B\}\]
Irisan (Intersection)
\[A \cap B = \{x \mid x \in A \text{ dan } x \in B\}\]
Selisih (Difference)
\[A - B = \{x \mid x \in A \text{ dan } x \notin B\}\]
Komplemen
\[\bar{A} = A^c = U - A = \{x \in U \mid x \notin A\}\]
Contoh 2.1

Misalkan \(U = \{1,2,3,4,5,6,7,8\}\), \(A = \{1,2,3,4\}\), \(B = \{3,4,5,6\}\). Maka:

  • \(A \cup B = \{1,2,3,4,5,6\}\)
  • \(A \cap B = \{3,4\}\)
  • \(A - B = \{1,2\}\)
  • \(\bar{A} = \{5,6,7,8\}\)

03

Hukum Identitas

Hukum identitas menyatakan bahwa operasi gabungan dengan himpunan kosong dan operasi irisan dengan semesta tidak mengubah himpunan asal.

Teorema 3.1
Hukum Identitas (Identity Laws)
\(A \cup \emptyset = A\)
\(A \cap U = A\)

Himpunan kosong \(\emptyset\) adalah elemen identitas untuk operasi gabungan, sedangkan semesta \(U\) adalah elemen identitas untuk operasi irisan.

Contoh 3.1

Jika \(A = \{2, 4, 6\}\) dan \(U = \{1,2,3,4,5,6,7,8\}\):
\(A \cup \emptyset = \{2,4,6\} \cup \{\} = \{2,4,6\} = A\) ✓
\(A \cap U = \{2,4,6\} \cap \{1,2,3,4,5,6,7,8\} = \{2,4,6\} = A\) ✓

Catatan
Analogi dengan bilangan real: \(a + 0 = a\) (identitas penjumlahan) dan \(a \times 1 = a\) (identitas perkalian). Dalam himpunan, \(\emptyset\) berperan seperti 0 untuk \(\cup\), dan \(U\) berperan seperti 1 untuk \(\cap\).
04

Hukum Dominasi

Hukum dominasi menyatakan bahwa operasi tertentu menghasilkan hasil yang "mendominasi" atau mengalahkan himpunan lainnya.

Teorema 4.1
Hukum Dominasi (Domination Laws)
\(A \cup U = U\)
\(A \cap \emptyset = \emptyset\)
Contoh 4.1

Jika \(A = \{1,3,5\}\) dan \(U = \{1,2,3,4,5,6\}\):
\(A \cup U = \{1,2,3,4,5,6\} = U\) — semesta mendominasi gabungan ✓
\(A \cap \emptyset = \emptyset\) — kosong mendominasi irisan ✓

05

Hukum Idempoten

Operasi himpunan pada dirinya sendiri menghasilkan himpunan yang sama.

Teorema 5.1
Hukum Idempoten (Idempotent Laws)
\(A \cup A = A\)
\(A \cap A = A\)
Perbandingan dengan Logika
Hukum ini sejajar dengan hukum idempoten dalam logika proposisional: \(p \vee p \equiv p\) dan \(p \wedge p \equiv p\).
06

Hukum Komplemen

Hukum komplemen mengatur hubungan antara himpunan dengan komplemennya dan hubungan dengan semesta serta himpunan kosong.

Teorema 6.1
Hukum Komplemen (Complement Laws)
\(A \cup \bar{A} = U\)
\(A \cap \bar{A} = \emptyset\)
\(\bar{\bar{A}} = A\)    (Hukum Involusi)
\(\bar{U} = \emptyset\)
\(\bar{\emptyset} = U\)
Contoh 6.1

Misalkan \(U = \{1,2,3,4,5\}\), \(A = \{1,2,3\}\), maka \(\bar{A} = \{4,5\}\).
\(A \cup \bar{A} = \{1,2,3,4,5\} = U\) ✓
\(A \cap \bar{A} = \emptyset\) ✓
\(\bar{\bar{A}} = \overline{\{4,5\}} = \{1,2,3\} = A\) ✓

07

Hukum Komutatif

Urutan himpunan dalam operasi gabungan dan irisan tidak memengaruhi hasilnya.

Teorema 7.1
Hukum Komutatif (Commutative Laws)
\(A \cup B = B \cup A\)
\(A \cap B = B \cap A\)
Perhatian
Hukum komutatif tidak berlaku untuk operasi selisih himpunan! Umumnya \(A - B \neq B - A\). Contoh: \(A = \{1,2,3\}\), \(B = \{3,4,5\}\), maka \(A - B = \{1,2\}\) tetapi \(B - A = \{4,5\}\).
08

Hukum Asosiatif

Pengelompokan himpunan dalam operasi gabungan dan irisan tidak memengaruhi hasilnya.

Teorema 8.1
Hukum Asosiatif (Associative Laws)
\((A \cup B) \cup C = A \cup (B \cup C)\)
\((A \cap B) \cap C = A \cap (B \cap C)\)
Contoh 8.1

Misalkan \(A=\{1,2\}\), \(B=\{2,3\}\), \(C=\{3,4\}\):
\((A \cup B) \cup C = \{1,2,3\} \cup \{3,4\} = \{1,2,3,4\}\)
\(A \cup (B \cup C) = \{1,2\} \cup \{2,3,4\} = \{1,2,3,4\}\) ✓

09

Hukum Distributif

Hukum distributif adalah salah satu hukum terpenting dan sering digunakan dalam penyederhanaan ekspresi himpunan.

Teorema 9.1
Hukum Distributif (Distributive Laws)
\(A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\)
\(A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\)
Contoh 9.1 — Verifikasi Distributif

Misalkan \(A=\{1,2,3\}\), \(B=\{2,4,6\}\), \(C=\{3,4,5\}\):
Sisi kiri: \(B \cap C = \{4\}\), maka \(A \cup (B \cap C) = \{1,2,3,4\}\)
Sisi kanan: \(A \cup B = \{1,2,3,4,6\}\), \(A \cup C = \{1,2,3,4,5\}\)
\((A \cup B) \cap (A \cup C) = \{1,2,3,4\}\) ✓

Analogi Aljabar
Hukum distributif himpunan analog dengan aljabar bilangan: \(a(b + c) = ab + ac\). Namun, di himpunan berlaku dua arah: \(\cup\) mendistribusikan atas \(\cap\) dan sebaliknya.
10

Hukum De Morgan

Hukum De Morgan adalah salah satu teorema paling terkenal dan berguna dalam teori himpunan. Dikemukakan oleh matematikawan Augustus De Morgan (1806–1871).

Teorema 10.1 — De Morgan
Hukum De Morgan (De Morgan's Laws)
\(\overline{A \cup B} = \bar{A} \cap \bar{B}\)
\(\overline{A \cap B} = \bar{A} \cup \bar{B}\)

Komplemen dari gabungan sama dengan irisan komplemen. Komplemen dari irisan sama dengan gabungan komplemen.

Contoh 10.1 — Verifikasi De Morgan

Misalkan \(U=\{1,2,3,4,5,6\}\), \(A=\{1,2,3\}\), \(B=\{2,3,4\}\).
Hukum pertama:
\(A \cup B = \{1,2,3,4\}\) → \(\overline{A \cup B} = \{5,6\}\)
\(\bar{A} = \{4,5,6\}\), \(\bar{B} = \{1,5,6\}\) → \(\bar{A} \cap \bar{B} = \{5,6\}\) ✓

Hukum kedua:
\(A \cap B = \{2,3\}\) → \(\overline{A \cap B} = \{1,4,5,6\}\)
\(\bar{A} \cup \bar{B} = \{4,5,6\} \cup \{1,5,6\} = \{1,4,5,6\}\) ✓

Generalisasi De Morgan

Hukum De Morgan dapat digeneralisasi untuk \(n\) himpunan:

\(\overline{A_1 \cup A_2 \cup \cdots \cup A_n} = \bar{A}_1 \cap \bar{A}_2 \cap \cdots \cap \bar{A}_n\)
\(\overline{A_1 \cap A_2 \cap \cdots \cap A_n} = \bar{A}_1 \cup \bar{A}_2 \cup \cdots \cup \bar{A}_n\)
11

Hukum Absorpsi

Hukum absorpsi (penyerapan) menyatakan bahwa ketika sebuah himpunan digabungkan atau diiriskan dengan ekspresi yang mengandung himpunan itu sendiri, hasilnya "diserap" menjadi himpunan tersebut.

Teorema 11.1
Hukum Absorpsi (Absorption Laws)
\(A \cup (A \cap B) = A\)
\(A \cap (A \cup B) = A\)
Bukti — \(A \cup (A \cap B) = A\)
(1)\(A \cup (A \cap B)\)
(2)\(= (A \cup A) \cap (A \cup B)\)   [Hukum Distributif]
(3)\(= A \cap (A \cup B)\)   [Hukum Idempoten: \(A \cup A = A\)]
(4)\(= (A \cup \emptyset) \cap (A \cup B)\)   [Hukum Identitas: \(A = A \cup \emptyset\)]
(5)\(= A \cup (\emptyset \cap B)\)   [Hukum Distributif]
(6)\(= A \cup \emptyset\)   [Hukum Dominasi: \(\emptyset \cap B = \emptyset\)]
(7)\(= A\)   [Hukum Identitas] ∎
□
Contoh 11.1

Misalkan \(A = \{1,2,3\}\), \(B = \{2,3,4,5\}\):
\(A \cap B = \{2,3\}\), maka \(A \cup (A \cap B) = \{1,2,3\} \cup \{2,3\} = \{1,2,3\} = A\) ✓

Metode Pembuktian Teorema

Ada beberapa metode yang lazim digunakan untuk membuktikan kesetaraan himpunan:

1. Metode Keanggotaan (Membership Method)

Tunjukkan bahwa setiap elemen pada sisi kiri juga ada di sisi kanan, dan sebaliknya. Gunakan jika dan hanya jika (\(\Leftrightarrow\)).

Contoh: Bukti Hukum De Morgan I — \(\overline{A \cup B} = \bar{A} \cap \bar{B}\)

Misalkan \(x\) sembarang elemen. Kita tunjukkan \(x \in \overline{A \cup B} \Leftrightarrow x \in \bar{A} \cap \bar{B}\).

→\(x \in \overline{A \cup B}\)
\(\Leftrightarrow x \notin (A \cup B)\)   [def. komplemen]
\(\Leftrightarrow \neg(x \in A \vee x \in B)\)   [def. gabungan]
\(\Leftrightarrow x \notin A \wedge x \notin B\)   [De Morgan Logika]
\(\Leftrightarrow x \in \bar{A} \wedge x \in \bar{B}\)   [def. komplemen]
\(\Leftrightarrow x \in \bar{A} \cap \bar{B}\)   [def. irisan] ∎
□

2. Metode Aljabar Himpunan

Transformasikan satu sisi persamaan ke sisi lainnya dengan menggunakan hukum-hukum yang sudah terbukti, langkah demi langkah.

Contoh: Bukti \(A \cup (A \cap B) = A\) (Hukum Absorpsi)
\(A \cup (A \cap B) = (A \cap U) \cup (A \cap B)\)   [Identitas]
\(= A \cap (U \cup B)\)   [Distributif]
\(= A \cap U\)   [Dominasi: \(U \cup B = U\)]
\(= A\)   [Identitas] ∎
□

Dualitas dalam Teori Himpunan

Perhatikan bahwa setiap teorema himpunan memiliki dual-nya, yang diperoleh dengan mengganti:
\(\cup \leftrightarrow \cap\),   \(\emptyset \leftrightarrow U\),   \(\subseteq \leftrightarrow \supseteq\)
Prinsip dualitas ini berarti: jika sebuah pernyataan tentang himpunan terbukti benar, maka pernyataan dualnya juga benar.
